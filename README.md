# Asuwaju Odusote Humanitarian Foundation (AOHF) Website

A complete, responsive humanitarian foundation website built with HTML, CSS, and vanilla JavaScript.

## 🎨 Features

### Design
- **Fully Responsive**: Mobile-first design that works on all devices
- **Modern UI**: Clean, professional humanitarian foundation aesthetic
- **Consistent Branding**: Unified color scheme and typography throughout

### Navigation
- **Sticky Header**: Navigation stays accessible while scrolling
- **Dropdown Menus**: Organized navigation with dropdown submenus
- **Mobile Menu**: Hamburger menu for mobile devices
- **Smooth Scrolling**: Enhanced user experience

### Functionality
- **Interactive Forms**: Contact and volunteer application forms
- **Image Gallery**: Lightbox functionality for gallery images
- **Counter Animation**: Animated statistics on scroll
- **Scroll to Top**: Convenient navigation button
- **Social Media Integration**: Links to all social platforms

## 🚀 Getting Started

1. **Add Images**: Place all required images in their respective folders
2. **Open in Browser**: Simply open `index.html` in a web browser
3. **No Build Process**: Pure HTML/CSS/JS - no compilation needed

## 🔧 Customization

### To Update Content:
1. Open the relevant HTML file
2. Locate the section you want to modify
3. Update text, images, or links as needed
4. Save and refresh browser

### To Change Colors:
1. Open `assets/css/styles.css`
2. Modify the CSS variables in the `:root` section
3. Save and refresh browser

### To Add New Pages:
1. Copy an existing page as a template
2. Update the content
3. Add navigation links in all pages
4. Update footer links
